# 📌 Project Overview

This project demonstrates an end-to-end Business Intelligence workflow for analyzing Airbnb listings across major European cities.

Instead of creating only a dashboard, the project follows the complete BI process:

- Data Collection
- Data Cleaning & Transformation
- Data Warehouse Design
- Star Schema Modeling
- Power BI Data Modeling
- DAX Measures
- Interactive Dashboard Development
- Business Insights

# 💡 Business Insights

- Amsterdam has the highest average listing price.
- Athens is the most affordable city.
- Entire Home/Apt is the most common room type.
- Listings closer to the city center generally have higher prices.
- Weekend prices are slightly higher than weekday prices.
- Guest satisfaction remains consistently high across all room types.
